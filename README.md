Forex News Impact Analyzer
The Forex News Impact Analyzer is a high-performance, web-based tool designed to automate the correlation between macroeconomic news events and historical price movements. By integrating high-granularity price data with verifiable economic calendars, it allows traders to visualize exactly how the market responded to specific news spikes with a single click.
+4

### Project Motivation
Manual cross-referencing between news aggregators and charting platforms is a significant time sink for professional and retail traders. This tool bridges the analytical gap by providing a dedicated, automated interface for strategy validation and historical back-testing.
+2

### Key Features

Automated Data Correlation: Fetches and aligns disparate data streams, including news timestamps and price action, into a unified visual dataset.


Interactive Visualization: Generates high-resolution charts with clear vertical markers indicating the precise release time of news events.
+1


High-Granularity Analysis: Supports assets across Forex and Cryptocurrencies, focusing on major catalysts like Non-Farm Payroll (NFP) and interest rate decisions.

Web-Based Accessibility: Built as a responsive single-page application compatible across all operating systems and devices.
